# CNN Project

This project contains CNN model implementations including:
- DenseNet Transfer Learning models
- MobileNet models  
- Custom CNN architectures
- BatchNorm implementations

## Models
The trained models are stored in the `models/` directory.

## Notebooks
- `main.ipynb` - Main implementation
- `main_jesus.ipynb` - Jesus's implementation
- `main_copy_laura.ipynb` - Laura's implementation
- `main copy_tl_laura.ipynb` - Transfer learning implementation
